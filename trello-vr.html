Kanban VR

This project was created for a Hackathon. We wanted to integrate office productivity tools into a VR environment. We made a virtual office space where multiple people can connect, meet and talk with actual voice, and move around a virtual Kanban board. The board is connected to Trello, and can send and receive information about the state of the website.
The user can drag cards to and from different panels to show the status of tasks.




Trello Integration

Using Trello's API, we can both grab data from the website, and update the website's cards through our application. Users in the online room can see the real-time state of the Kanban board in the virtual space. Ideally, we'd want to expand to integrate other work tools into this environment.

Real Time Communication

We wanted to be able to have an actual VR meeting, so users can connect to an online room, and watch other users move and interact with the board. The users can also talk to each other. This experience gives the users a sense that they are in the same space, even when everyone is working remotely.

Avatars

I made custom avatars for each participant. Since the hackathon was "Ragnarok" themed, I gave each person a Viking attire.
